## 特集

どんなときだって、どんなことだってぼくらは JavaScript と一緒にやってきた。
JavaScript 好きなら、きっとなんでも JavaScript でやってしまいたいはず。
今世の中で考えられている JavaScript よりも、さらに可能性をおし拡げることがテーマの記事を書いてもらえるとありがたい！(しかし、それ以外の記事も歓迎です。)

- JavaScript で DJ

- JavaScript で海外旅行
- JavaScript で USB にアクセス！

かなり強引でも OK です！
